# p2e router

Read these first:
- `workflows/p2e-policy.md`

Routing rules:
- requests about starting or mapping a project -> `workflows/p2e-bootstrap.md`
- requests about creating or filling a story -> `workflows/p2e-add-story.md`
- requests about implementing planned work -> `workflows/p2e-work-on-next.md`
- requests about label or issue reconciliation -> `workflows/p2e-sync-labels.md`
